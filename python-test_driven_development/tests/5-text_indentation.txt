Spacing Handling
================
It should trim spaces at the beginning of lines (and the text itself):
    >>> text_indentation("   Hello.   World")
    Hello.
    <BLANKLINE>
    World

It should handle delimiters with no spaces after them:
    >>> text_indentation("Hello.World")
    Hello.
    <BLANKLINE>
    World

It should handle multiple delimiters in a row (skip spaces in between if any):
    >>> text_indentation("Hello?   ...   World")
    Hello?
    <BLANKLINE>
    .
    <BLANKLINE>
    .
    <BLANKLINE>
    .
    <BLANKLINE>
    World

Edge Cases
==========
Empty string:
    >>> text_indentation("")

String with only spaces:
    >>> text_indentation("      ")

Text without delimiters:
    >>> text_indentation("Just a simple sentence")
    Just a simple sentence
